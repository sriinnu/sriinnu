I build AI systems that think in categories borrowed from Indian philosophy.
Not as metaphor — as actual architecture. Memory consolidation modeled on dream cycles.
Knowledge classification based on epistemological types from the Nyaya school.
Deliberation through formal syllogistic argument. Identity continuity across discrete sessions.

All of it runs on heuristics, not token burns. The LLM is a tool, not the brain.

### The Ecosystem

**[Chitragupta](https://github.com/sriinnu/chitragupta)** — The brain. A 17-package TypeScript monorepo for AI cognition: memory, identity, attention, affect, intention, self-reflection, deliberation, self-evolution. Exposes CLI, HTTP server, MCP server, and web dashboard. 12,000+ tests.

**[Takumi](https://github.com/sriinnu/takumi)** — The hands. Terminal coding agent with a custom renderer (not React/Ink), streaming agent loop, multi-agent orchestration, and deep Chitragupta integration for memory and predictions.

**[PAKT](https://github.com/sriinnu/clipforge-PAKT)** — The compression layer. Lossless prompt compression that saves 30-50% tokens on structured data. TypeScript library + CLI + MCP server + Chrome extension + Tauri desktop app. Published on [npm](https://www.npmjs.com/package/@sriinnu/pakt).

**[Kosha Discovery](https://github.com/sriinnu/kosha-discovery)** — The registry. Auto-discovers AI models, credentials, and pricing across Anthropic, OpenAI, Google, Bedrock, Vertex, Ollama, and OpenRouter.
